# Quickstart

1. Make sure pnpm is installed `npm i -g pnpm`
2. `cd` to project
3. Install dependencies `pnpm i`
4. Make sure the configuration service is running.
5. Run dev command `pnpm dev`
6. It will show you what to open in terminal.
